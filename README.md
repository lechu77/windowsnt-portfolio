# Windows 98 Interactive Portfolio

A retro Windows 98 inspired interactive resume built with React.

## Features

- Windows 98 desktop interface
- draggable windows
- playable Solitaire
- Winamp music player
- MS-DOS prompt with commands
- embedded Unity web game
- fake Internet Explorer

## Tech Stack

- React + Vite
- TailwindCSS
- Cloudflare R2 (music hosting)
- Cloudflare Pages (deployment)
- GitHub (source control)
